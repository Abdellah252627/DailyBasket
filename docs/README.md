# DailyBasket - Advanced Grocery Store Platform

A modern, fully-featured online grocery shopping platform built with HTML, CSS, and JavaScript. This project has evolved from a basic front-end application to a comprehensive e-commerce solution with advanced features including user authentication, dynamic cart management, real-time inventory tracking, and modern JavaScript architecture.

## üì¶ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/dailybasket.git
   cd dailybasket
   ```

2. **Open in browser:**
   - Open `Chitkara-Grocery-Store-main/HTML/index.html` in your web browser
   - No additional setup required - runs entirely in the browser

3. **For development:**
   - Use any modern web server (Apache, Nginx, or VS Code Live Server)
   - All dependencies are included - no npm/node required

## üéØ Usage Guide

### For Customers:
1. **Browse Products:** Explore 20+ products across 12 categories
2. **Search & Filter:** Use advanced search and filtering options
3. **Add to Cart:** Click "Add to Cart" on any product
4. **Manage Cart:** Update quantities, apply coupons, proceed to checkout
5. **User Account:** Register/login to save cart and preferences

### For Developers:
- **Database Admin:** Access admin panel at localhost for data management
- **Session Dashboard:** Monitor user sessions and analytics
- **Export Data:** Download product/user data as JSON/CSV
